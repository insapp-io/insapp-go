# insapp-go
Backend part of the Insapp project written in Go-lang

## Dependencies
```
cd src && go get
```

## Build & Launch

Edit config file

```
vi config.example.json
mv config.example.json config.json
```

Check that you have mongodb running

```
cd src && go build && ./src
```

Now your process is listening on 0.0.0.0:9000
