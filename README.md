# insapp-go
Backend part of the Insapp project written in Go-lang

## Dependencies
```
go get github.com/gorilla/mux
go get gopkg.in/mgo.v2
```

## Build & Launch

Check that you have mongodb running

```
cd src && go build && ./src
```

Now your process is listening on 0.0.0.0:9000
