Insapp

Comment publier ?

L'adresse de l'interface d'administration est la suivante : insapp.fr/admin.

Pour pouvoir publier depuis l'interface d'administration, un compte d'association est nécessaire. Si votre association n'en dispose pas encore, ou si vous avez oublié le mot de passe, n'hésitez pas à envoyer un mail à aeir-insapp@insa-rennes.fr. Nous répondons sous peu !

Une nouvelle interface d'administration est en cours de développement.

Pourquoi publier ?

Quels sont les avantages d'Insapp par rapport aux mails ?

  • L’information ayant attrait à la vie insalienne est centralisée et accessible via une API
  • Côté utilisateur, le système d’événements permet de ne rien manquer
  • Côté utilisateur, l’interaction est possible et profitable à tous grâce aux commentaires

Quels sont les avantages d'Insapp par rapport à Facebook ?

  • L’information ayant attrait à la vie insalienne est centralisée et accessible via une API
  • L’information est stockée localement, et nous en avons le contrôle
  • L'information n'est pas noyée dans Entraide INSA
  • Toutes les associations ont la même visibilité
  • Côté utilisateur, pas besoin d’un compte Facebook
